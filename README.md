# LedgerLens Webapp

A modern AI-powered financial analysis platform for generating intelligence reports from portfolio documents.

## Features

- **Document Analysis**: Upload financial documents (PDF) for instant AI analysis.
- **Intelligence Reports**: Detailed breakdown of risks, sentiment, and key highlights.
- **AI Accuracy Scores**: Track the confidence and accuracy of AI-generated insights.
- **Interactive Dashboard**: Overview of all analyzed portfolios with sentiment mapping.
- **Search & Retrieval**: Semantic search across document contents.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI, Lucide Icons, Shadcn/ui
- **State Management**: React Hooks
- **API Integration**: Axios

## Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

### Running the App

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Project Structure

- `/app`: Next.js App Router pages and layouts
- `/components`: Reusable UI components
- `/lib`: Utility functions and API clients
- `/public`: Static assets (images, icons)
